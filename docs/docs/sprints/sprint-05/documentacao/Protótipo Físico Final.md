# Protótipo Físico Final

---

## 1. Objetivo desta documentação

&nbsp; Esta página complementa a documentação da sprint final com o registro visual do protótipo físico de cada módulo após a evolução até o estado entregue. As imagens permitem:

- **Localizar e compreender** como cada parte do sistema se apresenta na montagem final;
- **Inspecionar visualmente** a construção e identificar possíveis inconsistências entre o que foi documentado e o que foi montado;
- **Servir de referência** para outras equipes que queiram continuar o desenvolvimento, reutilizando ideias já implementadas e executadas no hardware.

&nbsp; As fotos abaixo correspondem ao estado final de cada módulo nesta sprint. Os arquivos originais estão em `docs/static/img/fotos-protótipo-físico-final/`.
